# 🛒 E-commerce Website Project (HTML & CSS)

## 📚 Project Context

This project was assigned as part of my first-year (L1) coursework at SUPINFO.

The objective was to individually design and develop a complete e-commerce website for a non-profit organization (e.g., WWF, Handicap International, SPA, etc.) using only HTML and CSS.

The focus of the project was not only on functionality but also on:
- Code quality
- Visual design
- Accessibility
- Responsive layout

---

## 🎯 Project Objectives

The website includes:

### 🏠 Home Page
- Functional navigation bar
- Centered logo
- Hero section with overlay text
- Featured products
- Highlighted categories
- Footer with links, social icons, and legal information

### 📂 Category Pages (Minimum 4)
- Product grid layout
- Product images
- Product names and prices
- Links to product detail pages

### 📦 Product Detail Pages (Minimum 8)
- Multiple product images
- Detailed description
- Product characteristics
- Quantity selector
- Add to cart button
- Suggested similar products

### 🛍 Support / Cart Page
- Functional contact form
- Email message reception

### ❌ 404 Error Page
- Custom error message
- Redirection button to homepage

---

## 💻 Technical Requirements

- Fully responsive design (Desktop, Tablet, Mobile)
- Accessible content (RGAA / WCAG principles)
- No broken links
- Clean and well-structured HTML & CSS
- Organized file architecture

---

## 🎨 Design & UX Focus

- Consistent typography and color palette
- Clear content hierarchy
- Balanced layout
- Simple and intuitive navigation
