1. Introduction du projet
Dans le cadre de votre premier projet, vous êtes en charge individuellement de réaliser la construction d'un site de vente en ligne pour une association (WWF, Handicap International, SPA, etc.). Il est attendu à ce que vous conceviez un site web complet (HTML + CSS) en faisant attention à la fois à la qualité du code mais aussi au résultat visuel final.

2. Système de notation
Le rendu sera noté sur un total de 100 points qui se partageront entre l’ergonomie, le design ainsi que l’accessibilité du contenu. Le détail de la notation sera développé dans chaque partie.
